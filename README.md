# Azamjon Ergashev's Portfolio

Welcome to the personal portfolio of Azamjon Ergashev, a Frontend Developer. This project showcases my skills, projects, and contact information.

## Project Structure

```
azamjon-portfolio
├── index.html          # Main HTML document for the portfolio website
├── assets
│   ├── images          # Directory for images (profile pictures, project screenshots)
│   └── icons           # Directory for icons (social media icons, decorative elements)
├── js
│   └── script.js       # JavaScript logic for handling form submission and Telegram bot integration
├── css
│   └── style.css       # Additional styles (if needed, primarily uses Tailwind CSS)
└── README.md           # Documentation for the project
```

## Features

- **Responsive Design**: The portfolio is fully responsive, adapting to mobile, tablet, and desktop views.
- **Dark Mode**: A modern dark mode design enhances user experience.
- **Smooth Animations**: Includes hover effects, fade-ins, and slide-ups for a dynamic feel.
- **Contact Form**: Users can reach out via a contact form that integrates with a Telegram bot for notifications.
- **Skills and Projects**: Displays skills and projects in an engaging layout with hover animations.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd azamjon-portfolio
   ```
3. Open `index.html` in your preferred web browser to view the portfolio.

## Technologies Used

- HTML
- Tailwind CSS (via CDN)
- Vanilla JavaScript
